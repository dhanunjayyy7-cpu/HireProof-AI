# HireProof

HireProof is an AI-powered platform that helps students, freshers, and job seekers detect fake job and internship scams.

## Features

* Analyze suspicious job offers
* Detect scam signals
* Trust score
* Red flags
* Reality check
* Clean result summary

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Supabase
* Claude AI / AI Gateway
